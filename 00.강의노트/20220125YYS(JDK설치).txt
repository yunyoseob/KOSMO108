jdk 설치
------------------------
1. jdk-8u202-windows-x64.exe 설치하기

2. 설치 디렉토리
	C:\Program Files
	개발 엔진
		C:\Program Files\Java\jdk1.8.0_202
	실행 엔진
		C:\Program Files\Java\jre1.8.0_202
	
3. 자바 환경변수 세팅하기
	C:\Program Files\Java\jdk1.8.0_202\bin\javac.exe
	javac.exe 컴파일러를 이 컴퓨터의 어느 디렉토리에서든지 실행되게 하기 위해서
	환경변수를 설정한다.

	내컴퓨터-우클릭- 속성 - 설정
	c:\ 에서는 안 되고 C:\Program Files\Java\jdk1.8.0_202\bin\javac.exe에서만
	되는 문제 해결


4. 이클립스 사용
	이클립스를 설치해서 jdk 플러그인 한다.

JAVA 설치시 경로가 jre로 되어있는데, 일단 설치해도 된다.
단, 설치시 jre로 path가 지정되어 있는지, 아닌지는 꼭 알아야 한다.

제어판에 어떻게 설치가 되어 있는지, 시스템 엔지니어 담당의 일을 구체적으로 알 필요는 없다.
그러나, 그게 뭔지 알아야 내가 할 수 있는 것인지 없는 것인지 알 수 있기 때문에, 알아야 할 필요가 있다.

Directory가 꼬이면 큰 일나기 때문에 이런 걸 잘 체크해야 한다.

console 명령어 (cmd)
```
cd dir cls .(도트 연산자) * / 
```
정말 중요하다. 꼭 알아야 한다.
잘 모르겠으면 help를 붙이면 된다.
단, .(도트연산자)와 *(별표)는, / 는 help붙여도 아무것도 안 나온다.
잘 모르겠으면 help를 눌러서 help 뒤에 뭐가 오는지 꼼꼼하게 보셈.
(cmd에서 help 치면 배치명령어가 나오는데 잘 확인해봐야함)

---------------
cd: change directory
도움말 보기: help cd

dir: directory

cls: clear 


.: 도트 연산자, 링크 연산자
console 에서 .(도트)는 : 현재 디렉토리 또는 링크 연산자로 사용한다.
.: 현재 디렉토리
..: 상위 디렉토리

cd: 체인지 디렉토리
도움말 보기: c:\>help cd

console에서 디렉토리를 찾아 갈 때는
	*: 미풍(조미료 이름임)표시: 아스타리스크: 별표 : 와일드카드 : 곱하기 연산자 : 카디널리티 : 전체 표시 역할
	cd P*s (x86)
	cd Program Files (x86)
앞으로 전진: 디렉토리 이름 쓰기
뒤로 가기: .(도트 연산자) 사용하기
	cd/ 그 디렉토리의 최상위 디렉토리(루트)로 가세요.
C:\>javac -> 실행 안 됨
C:\>javac.exe -> 실행 안 됨
C:\>java.exe -> 실행 됨
C:\>Java.exe -> 실행 됨
