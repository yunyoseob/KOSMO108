폼태그 연습

    form 태그 속성
  1. name : 자바스크립트로 폼을 제어할 떄 사용하는 폼 이름 지정
  2. id : 자바스크립트로 폼을 제어할 때 사용하는 폼 이름 지정
  3. method : 폼 요소 내용을 서버쪽 프로그램에 넘겨주는 방식 지정(GET, POST)
  4. action : form 태그 안의 내용을 처리해 줄 서버 프로그램을 지정(#은 꼼짝마)
  • fieldset, legend는 폼 요소를 그룹별로 묶는다.
