	Object
	String
	Integer
	System
	Warapper
java.util.*
	데이터(컬렉션 프레임워크), 날짜, 시간 관련된 클래스 묶음

	HashMap
	Date
	Calendar
java.math.*
	숫자에 관련된 클래스 묶음
	BicDecimal

	파일 및 입출력에 관련된 클래스 묶음
	File
	inputStream
	outputStream

	JDBC 를 통한 데이터베이스 연동 클래스 묶음
	DriverManager
	Connection
	Statement
	PreparedStatement
