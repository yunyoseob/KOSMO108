The border-style 속성

A dotted border. 테두리를 점선으로 표시

A dashed border. 테두리를 짧은 직선으로 표시

A solid border. 테두리를 실선으로 표시

A double border. 테두리를 이중선으로 표시, 두 선 사이의 간격이 border-width

A groove border. 테두리를 홈이 파인 듯 입체 느낌

A ridge border. 테두리를 창에서 튀어나온 것처럼 표시

An inset border. 표에서 boder-collapse:seperate일 경우 전체 박스 테두리가 창에 박혀 있는 것처럼 표시

An outset border. 표에서 vorder-collapse:seperate일 경우 전체 박스 테두리가 창에서 튀어나온 것처럼 표시

No border. 테두리가 없음. 기본값

A hidden border. 테두리를 감춤. 표에서 border-collapse일 경우 다른 테두리도 표시하지 않음

A mixed border.
